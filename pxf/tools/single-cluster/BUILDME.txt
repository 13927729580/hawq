Build
=====

parameters
PXF_ARTIFACTS -		directory for PXF tarballs relative to tars subdirectory
					default ../../../build/distributions
					optional
PHD_VERSION		-	PHD stack version to use (e.g. 2.2)
					mandatory

make refresh_tars PHD_VERSION=x PXF_ARTIFACTS=y
make clean all

Note: You have to do clean
